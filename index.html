Offline support

Download instructions

  1. Visit this site in Chrome, Edge or Safari.
  2. Tap the Download app button when it lights up.
  3. Accept the prompt to add it to your home screen.

Don’t see the download prompt?

The button appears when the browser confirms the site meets PWA install criteria (served over HTTPS, manifest, service worker). On desktop, check the address bar icon Install nextwave.

Manual add to home screen
  • iOS Safari: Share → Add to Home Screen
  • Android Chrome: ⋮ menu → Install app
  • Desktop Chrome/Edge: Install icon at right of address bar
